# 서울시 노인 보행안전 사각지대 탐지 모델

사고 위험과 보행 노출은 높지만 안전 인프라가 상대적으로 부족한 지역을 정량적으로 식별하는 점수 모델입니다.
**2026 서울시 빅데이터 활용 경진대회 분석부문** 출품작 (팀명: 가드레일)

---

## 프로젝트 개요

서울시는 매년 노인보호구역을 확대하고 있지만, 노인 보행 사고의 99%는 보호구역 밖에서 발생하고 있습니다. 기존 정책은 '사고가 많이 난 곳'을 중심으로 사후 대응하는 방식이라, 앞으로 사고가 날 가능성이 높은 지역을 미리 식별하기 어렵습니다.

이 프로젝트는 단순 사고 발생지가 아니라, **'보행 충돌 노출은 높지만 안전 인프라가 상대적으로 부족한 지역'을 사각지대로 정의**하고, 이를 정량적으로 점수화하는 모델을 설계했습니다.

## 분석 과정

### 1. 데이터 전처리 및 통합

원본 데이터의 단위와 변수 형식이 서로 달라, 자치구명을 기준으로 정제한 뒤 하나의 분석 테이블로 머지했습니다.

- 10m 도로구간 유동인구 데이터에서 낮 시간대 노인 유동을 추출해 `노인유동인구_낮평균` 변수 생성 (거주인구가 아닌 실제 보행 노출 규모 반영)
- 횡단보도 데이터로부터 자치구별 횡단보도수 집계 및 보행등·음향신호기·고원식 설치율 산출
- 회귀분석용 표준화, 점수 모델용 정규화(0~1), 비교용 Min-Max(0~100) 3단계 스케일링

### 2. 회귀분석으로 변수 영향력 도출

`노인부상자수`를 종속변수로, 7개 변수를 설명변수로 회귀분석을 수행했습니다. (R² = 0.7484)

- 횡단보도수의 표준화 Beta(77.9)가 가장 컸습니다. → 노인 사고가 단순 인구 규모보다 '차량과 보행자가 만나는 물리적 접점'에 더 강하게 연결됨
- 음향신호기 설치율은 음의 Beta(-23.6) → 안전 보조 시설이 사고를 완화하는 보호 요인일 가능성 (단, 인과가 아닌 상관 해석)

### 3. 표준화 Beta 기반 가중치 산정

기존 점수 모델은 가중치를 분석가의 정책적 판단으로 설정해 주관성이 개입했습니다. 본 분석에서는 회귀에서 나온 표준화 Beta의 절댓값을 정규화해 가중치로 사용했습니다.

```
변수 가중치 = |표준화 Beta| / Σ|표준화 Beta|
```

### 4. 사각지대 점수 모델 설계

사고 위험·충돌 노출은 위험 증가 요인, 안전 인프라는 위험 완화 요인으로 보고 설계했습니다.

```
사각지대점수 = (사고위험점수 + 보행충돌노출점수) / (안전인프라점수 + 0.1)
```

> 분모의 `+0.1`은 안전 인프라 점수가 0인 지역에서 분모가 0이 되는 수리적 오류를 막기 위한 안정화 상수입니다. 최종 점수는 자치구 간 비교가 쉽도록 0~100으로 변환했습니다.

### 5. 자치구 → 행정동 드릴다운

자치구 단위 결과를 실제 정책 집행 단위로 구체화하기 위해, 상위 위험 자치구 내부를 행정동 단위로 재집계해 세부 개선 후보지를 도출했습니다.

## 주요 결과

- 사각지대 점수 1위는 **중구(100점)**. 주목할 점은 중구의 노인부상자수(188명)가 강남구(494명)·송파구(427명)보다 훨씬 적다는 것입니다.
- 단순 사고 건수로 우선순위를 매겼다면 강남구·송파구가 1순위였겠지만, 보행 노출 대비 안전 인프라 수준을 함께 보면 우선순위가 달라집니다.
- '현재 위험(실제 부상자수)' vs '미래 위험(사각지대 점수)'을 비교한 결과, 중구는 현재 위험은 낮지만 미래 위험이 가장 높은 잠재 위험지역으로 식별되어, 모델이 사후 대응이 아닌 **선제적 식별 도구**로 작동함을 확인했습니다.
- 행정동 드릴다운 결과 중구 신당동이 최우선 점검 대상으로 도출되었습니다.

## 한계와 추가 검증이 필요한 부분

1. **표본 크기** — 자치구 25개 표본에 설명변수 7개라 자유도가 빠듯합니다. 행정동 단위 확장이나 시계열 추가로 모델 안정성을 보강할 수 있습니다.
2. **인과 vs 상관** — 음향신호기의 음의 영향은 통계적 상관일 뿐, 사고 다발 지역에 사후적으로 더 설치된 역인과 가능성이 있습니다.
3. **가중치의 데이터 의존성** — 표준화 Beta 기반 가중치는 분석 시점 데이터에 의존하므로, 데이터 갱신 시 재산정이 필요합니다.
